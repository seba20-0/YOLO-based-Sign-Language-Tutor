# YOLO-based-Sign-Language-Tutor
## Installation
### `Step 1` - Clone the repository
```bash
git clone https://github.com/seba20-0/YOLO-based-Sign-Language-Tutor.git
```

### `Step 2` - Install requirements
```bash
pip install -r requirements.txt
```

### `Step 3` - Run the API
```bash
python app.py
```
